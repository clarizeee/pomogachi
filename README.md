Features:

- Ability to add tasks and subtasks
- Ability to add them to a queue and assign them for pomodoro
- Ability to edit exisitng tasks
- Ability to reassign queue items
- Ability to change pomodoro modes
- Ability to filter tasks based on category
- Easy deadline setup using datetime picker (i think)

Room for improvement:

- Actual pomodoro cycling
- Pomodoro mode edit
- user-level GIF editing
- A Less Janky Layout
- Ability to drag subtasks outside their parent and become standalone
- Ability to drag a standalone task and become a subtask
- Web compatibility and cloud storage (currently the saving is only local :( )
- homescreen widgets that can display current task in pomodoro/display tasks due today
